# Codebook

## Features:
* SubjectID
  The ID of the Subject observed
* Activity
  The activity observed. Can be one of:
  * WALKING
  * WALKING_UPSTAIRS
  * WALKING_DOWNSTAIRS
  * SITTING
  * STANDING
  * LAYING
